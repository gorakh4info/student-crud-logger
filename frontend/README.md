# Student CRUD Logger App

## Features
- CRUD operations (Create, Read, Update, Delete)
- Data stored in localStorage
- Centralized logger (logs stored in localStorage)
- Intentional bugs for debugging practice

## Known Issues
- students initialized as null
- crash on first add
- update creates new ID
- no validation for age
- logs only in console + localStorage

## Run Project
npm install
npm start

## Logs
Check logs in browser:
Application → Local Storage → logs
